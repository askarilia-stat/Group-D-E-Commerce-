## Deskripsi Project

Sistem basis data Marketplace/E-Commerce yang dibangun menggunakan MySQL untuk mendukung pengelolaan data produk, pelanggan, penjual, transaksi, pembayaran, pengiriman, dan ulasan produk. Proyek ini dibuat sebagai tugas akhir mata kuliah Basis Data dengan menerapkan konsep ERD, normalisasi, relasi antar tabel, View, Stored Procedure, Trigger, dan Function.
